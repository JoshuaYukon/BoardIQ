# BoardIQ Project

A Jira-like board application built with Ruby on Rails.

## Features

- User authentication with Devise
- Projects management
- Issues tracking with status (Todo, In Progress, Done)
- Boards for organizing issues

## Setup

1. Install Ruby and Rails
2. Run `bundle install`
3. Run `rails db:migrate`
4. Run `rails server`

## Usage

- Create projects
- Add boards to projects
- Create issues and assign to boards
- View the board with issues in columns
